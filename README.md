# Personal Portfolio

## Browser as a Framework
After many versions and redesign, I decided to go with something **very** simple.

I didn't want to follow the hype of whatever JavaScript framework is popular right now.
So a friend of mine, [Daniel Falbo](https://www.github.com/flbmdn), introduced me to a very cool and surely **out of the box** mentality.

Here's a link to the article: https://www.html3000.dev

I tried to keep the website as lightweight as possible with no Junk JS, CSS animations and other things that might slow it down.
That's it. A personal portfolio. Not a work of art, neither a blank page with some text on it.
